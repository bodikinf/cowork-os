# Example workspace — Yempik

> **EN —** A real `cowork-os` in the wild: Yempik's own workspace, **sanitized** (no client data, financials, or third-party names). Browse it to see what a *filled* system looks like — the templates in the repo are the empty version of these files. The guided installer (`INSTALL.md`) also reads this folder as a few-shot reference, so the workspace it generates for you matches this level.
> **IT —** Un `cowork-os` reale: il workspace di Yempik, **sanitizzato** (niente dati cliente, finanziari o nomi di terzi). Sfoglialo per vedere com'è un sistema *compilato* — i template nel repo sono la versione vuota di questi file.
